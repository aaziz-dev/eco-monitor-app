# 🌿 EcoMonitor – Android Environmental Monitoring App

## 📌 Overview

EcoMonitor is a mobile Android application designed to monitor wildlife species and environmental conditions in natural parks.
The app allows real-time tracking of ecological data and sends alerts when environmental thresholds are exceeded.

---

## ⚙️ Tech Stack

* Android (Java/Kotlin)
* SQLite (Local Database)
* Firebase (Cloud Sync)
* GPS & Geolocation APIs
* Device Sensors (Temperature, Humidity)

---

## 🚀 Features

* 📊 Manage species and habitats (name, conservation status, population)
* 🌡️ Real-time environmental data collection (temperature & humidity)
* 📍 GPS tracking and geocoding (convert coordinates to readable addresses)
* 🔔 Alerts & notifications when ecological thresholds are exceeded
* 💾 Offline mode with local database (SQLite)
* ☁️ Cloud synchronization using Firebase
* 🌐 Multilingual support (English & French)

---

## 🧠 System Design

* Local persistence with SQLite for offline access
* Firebase integration for remote data storage
* Sensor-based data collection
* Threshold-based alert system
